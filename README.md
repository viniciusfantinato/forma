# Forma

Este repositório contém um projeto simples desenvolvido com o objetivo de **praticar e aplicar conceitos básicos de HTML5 e CSS3**. A proposta é construir uma página web estática que permita explorar técnicas fundamentais de estruturação e estilização de páginas. 

## 🎯 Objetivo

O objetivo principal desse projeto é consolidar conhecimentos iniciais de desenvolvimento web, focando na **estrutura semnântica do HTML5** e na **estilização com CSS3**, utilizando boas práticas e organização básica de código. 

Este projeto é voltado principalmente para **estudo, prática e experimentação**, servindo como base para quem está iniciando no desenvolvimento front-end.

## 🧠 Conceitos aplicados

Durante o desenvolvimento desse projeto, são utilizados os seguintes conceitos:

### HTML5
- Estrutura básica de um documento HTML
- Uso de **tags semânticas**
- Inserção de **imagens**
- Utilização de **links (âncoras)** para navegação
- Organização de conteúdo em seções

### CSS3
- **Box model**
- **Estilização de caixas**
- Aplicação de **cores**
- Uso de **fontes**
- Controle de **espaçamentos (margin e padding)**
- Ajustes básicos de layout

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3

## 📚 Finalidade educacional

Este repositório foi criado exclusivamente para **fins educacionais**, com o intuito de praticar fundamentos do desenvolvimento web antes de avançar para conceitos mais complexos como:

- Flexbox e Grid
- JavaScript
- Frameworks e bibliotecas front-end
